# Timidfox

A Firefox userChrome.css theme — dark, minimal, Zen/Arc-inspired with native vertical tabs and auto-hide navbar.

Modular CSS in `chrome/zen-modules/`. Drop into your Firefox profile's `chrome/` directory and enable `toolkit.legacyUserProfileCustomizations.stylesheets` in `about:config`.

See `chrome/userChrome.css` for the full module list and configuration notes.
